Advanced DeploymentÂ¶

We will be covering the following topics in this section:

  1. If more users have access to bots-monitor, use limited rights for users.
  2. If you use bots-monitor over the internet/outside your LAN, use HTTPS/SSL connection.
  3. Use MySQL or PostgreSQL as the Bots database.
  4. Use AS2 as a communication method.
  5. Bots has options to push changes from test to production.
  6. Information about performance.
